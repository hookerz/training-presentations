What is Polymer

Polymer is an open-source JS library for building web apps using web components…

Introduced by Google developers in May 2015 and has been gaining in popularity, especially with the release of 1.0 in late 2015.

Defining terms

Web components are a set of new features in the HTML and DOM specs for creating custom reusable widgets or components.

4 Main features: Custom Elements, Shadow DOM, HTML Imports, HTML Templates

Features

Polymer provides a number of features over vanilla web components:

  • Simplified way of creating custom elements
  • Two-way data binding
  • Computed properties
  • Conditional and repeat templates
  • Gesture events
  • Library of elements

Opinionated

  • There is a "right" way of doing things
  • Predictable code structure and style
  • Thoughtful choices about how to share data between components

Encapsulation

  • Easier collaboration / division of labor
  • DRY
  • QA and testing

Polymer Element Catalog

https://elements.polymer-project.org/

Drawbacks

  • Limited web components support; must use polyfill (IE11+)
  • Build tools are moderately difficult to configure
  • Smaller community than Angular/Backbone/etc
